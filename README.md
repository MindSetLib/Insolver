# InsuranceScoring
Mindset insurance scoring - product repository

# Установка пакета `insolver`
Запускается из папки с гит репозиторием
- обычная установка
```shell script
pip install .
```
- установка в режиме редактирования (можно изменять код самого пакета без его переустановки)
```shell script
pip install -e .
```
